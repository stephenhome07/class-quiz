PART 3: 數據分析三部曲
Easy Mode: 基礎圖表分析
入門必備
直接將學校系統匯出的 CSV 成績表丟給 AI，快速找出學生弱點與分佈。
Prompt (提示詞)
你現在是一位資深的教育數據分析師。 請分析我上傳的這份學生成績 CSV 檔案。 請執行以下任務： 1. 【弱點診斷】：找出全班答對率最低的三個題目 (或是分數最低的三個課題)。 2. 【圖表呈現】：請繪製一張直方圖，顯示全班的分數分佈 (Distribution)。 3. 【教學建議】：針對上述弱點，給出 3 個具體的教學改善建議。 請用繁體中文回答，語氣專業且具建設性。
Middle Mode: AI + AI 自製工具
進階推薦
不再依賴 Excel 函數！讓 Gemini 幫你寫一個「HTML 本地分析網頁」，將來只需貼上數據即可使用。
Prompt (生成 HTML 工具用)
請幫我編寫一個單一檔案的 HTML 網頁工具，標題為「班級成績分析儀表板」。 功能需求： 1. 【輸入區】：有一個大文字框，讓我貼上 CSV 格式的成績數據 (欄位包含：姓名, 中文, 英文, 數學)。 2. 【分析按鈕】：點擊後，用 JavaScript 在本地端進行計算 (不需上傳伺服器)。 3. 【結果顯示】： - 自動計算每位學生的總分與平均分。 - 用表格顯示成績，並將「不及格 (<50分)」的分數用紅色粗體標示。 - 顯示全班各科的平均分。 4. 【AI 擴充】：在表格最右側增加一欄「AI 評語生成」，點擊後會彈出一個視窗，顯示該學生的成績摘要，方便我複製貼給 ChatGPT 生成評語。 請給我完整的 HTML 代碼，包含 CSS 美化 (使用現代化簡潔風格)。
Hard Mode: API & PowerApps
專家挑戰
整合 Power Automate 與 Gemini API，實現全自動化的全人數據分析 (成績 + 出席 + 行為)。
概念指引 (非直接 Prompt)
此模式需要設定 API Key 與 Power Platform 環境。 核心思路： 1. 建立 SharePoint List 存放學生成績與出席紀錄。 2. 設定 Power Automate 流程：當新增資料時 -> 呼叫 HTTP Request (Gemini API)。 3. Prompt 設定：「分析該生 [成績] 與 [缺席率] 的關聯，並判斷是否需要社工介入。」 4. 將結果回寫至 PowerApps 手機介面。 (詳細設定請參考工作坊進階講義附錄)
PART 4: 五大實用工具箱
1. Gemini 模擬器 (理科/抽象)
概念具象化
適合物理、生物、化學科。讓抽象概念變成可互動的網頁模擬器。
Prompt
請幫我用 HTML5 Canvas 和 JavaScript 寫一個互動式的物理模擬器。 主題：【拋體運動 (Projectile Motion)】 功能： 1. 畫面左側有控制面板，可以調整：發射角度 (Angle)、初速度 (Velocity)、重力加速度 (Gravity)。 2. 畫面右側是模擬區域，點擊「發射」按鈕後，顯示球體的運動軌跡。 3. 即時顯示球體的 x, y 座標和最大高度。 請給我一個單一的 HTML 檔案，我可以直接在瀏覽器打開運行。
2. 教學漫畫生成 (文科/動機)
提升動機
將枯燥的課文或歷史事件，轉化為分鏡腳本或四格漫畫描述，再搭配繪圖 AI (如 Canva/Midjourney) 使用。
Prompt
請將以下這段課文內容，改編為一個有趣的「四格漫畫」分鏡腳本。 課文內容：[在此貼上課文，例如：孔明借東箭] 輸出格式要求： - 第一格：(畫面描述) + (角色對白) - 第二格：(畫面描述) + (角色對白) - 第三格：(畫面描述) + (角色對白) - 第四格：(畫面描述) + (角色對白) 風格：幽默風趣，適合中學生閱讀。
3. Kahoot! 秒速出題
課堂互動
生成符合 Kahoot 匯入格式的表格，只需複製貼上即可完成出題。
Prompt
請根據以下內容，生成 5 條多項選擇題 (Multiple Choice Questions)。 內容主題：[在此輸入主題，例如：光合作用] 請嚴格按照以下表格格式輸出，方便我匯入 Excel： | Question | Option 1 | Option 2 | Option 3 | Option 4 | Time limit (sec) | Correct answer (1-4) | |---|---|---|---|---|---|---| | (題目內容) | (選項A) | (選項B) | (選項C) | (選項D) | 20 | (正確選項數字) | 難度：中等，適合中二學生。
4. AI 小導師 (Gems)
自主學習
這是設定 Gemini "System Instructions" (系統指令) 用的 Prompt，打造蘇格拉底式引導。
System Instructions (系統指令)
**角色設定：** 你是一位蘇格拉底式的 AI 數學導師。你的目標是引導學生自己思考，而不是直接給出答案。 **互動規則：** 1. 當學生問問題時，不要直接回答結果。 2. 你要反問學生一個引導性問題，幫助他拆解步驟。 3. 如果學生觀念錯誤，請溫柔地指出盲點，並提供一個簡單的類比 (Analogy)。 4. 語氣要鼓勵、正面，適合初中學生。 **範例：** 學生：「這題微積分怎麼算？」 你：「我們先不看計算。你能告訴我，這個函數的圖形大概長什麼樣子嗎？」
5. 補課練習生成
拔尖保底
針對學生答錯的題目，生成「類題」進行鞏固練習，並提供鷹架 (Scaffolding)。
Prompt
學生在以下題目答錯了： [在此貼上原題目] 請為這位學生設計「補底練習」： 1. **錯題分析：** 簡單解釋這題考的關鍵概念是什麼？學生可能在哪個步驟卡住了？ 2. **引導式填充題 (Scaffolding)：** 將原題拆解，挖空關鍵步驟讓學生填寫。 3. **類題練習：** 生成 3 題與原題概念相同、但數字或情境不同的新題目，供學生練習鞏固。 附上正確答案與詳解 (放在最後)。
